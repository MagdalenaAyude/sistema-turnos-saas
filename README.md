# 📅 SISTEMA DE TURNOS SAAS

Sistema de gestión de turnos tipo SaaS diseñado para negocios locales como estética, peluquerías y servicios profesionales. Permite organizar, visualizar y administrar citas de forma simple y eficiente mediante un calendario interactivo.

---

## 🚀 OBJETIVO DEL PROYECTO

Digitalizar la gestión manual de turnos en pequeños negocios, reduciendo errores, mejorando la organización y optimizando el tiempo de atención.
Este proyecto nace como la evolución de un planner personal, reorientado a un producto con potencial comercial real.

---
